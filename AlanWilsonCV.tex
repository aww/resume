\section{Ph.D. thesis}
\cvline{title}{\emph{Search for a Supersymmetry Signature with the $Z\gamma$ plus Missing Transverse Energy Final State Using the D\O\ Detector}}
\cvline{supervisor}{Prof. Bing Zhou}
\cvline{description}{\small
Over the last four years the D\O\ detector at Fermilab's Tevatron Collider has collected 
$6.2$~fb$^{-1}$ of 1.96~TeV proton-antiproton collisions.
Using this data I search for new physics producing a photon,
a pair of electrons or muons from a $Z$ boson decay,
and missing transverse energy produced by undetected particles.
It is the first time that this signature has been searched for in a hadron collider experiment.
Certain gauge mediated supersymmetry breaking (GMSB)
theories would be observable via a few events of this type.
I conclude that the observed data is constitent with the standard model prediction,
no evidence for a supersymmetric signal is found.
Employing multivariate selection techniques (boosted decision trees)
and combining the analysis results from both electron and muon channels
we exclude the GMSB model at 95\% confidence level for $70 < \Lambda < 117.5$~TeV,
which corresponds to neutralino masses of
$111 < \tilde{\chi}_1^0 < 222$~GeV (the next-to-lightest superparticle).
}
